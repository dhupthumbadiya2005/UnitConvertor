# Unit Converter App

This Android application serves as a simple unit converter, allowing users to convert between different units of length such as centimeters, meters, feet, and inches.

## Features

- Conversion between various length units (Centimeters, Meters, Feet, Inches)
- Interactive UI with dropdown menus for selecting input and output units
- Real-time conversion as you input values
- Provides accurate results up to two decimal places
